# Dotfiles

My dotfiles.

**Important: `setup.sh` may not work successfully on another platform!**

> `setup.sh` is stable on fedora, MaxOSX

## Setup

```sh
sh setup.sh
```

## Current

- `setup.sh` for setting up
- `.vimrc` for vim
- `.zshrc` for zsh
- `.mytheme.sh` for powerlevel9k theme
