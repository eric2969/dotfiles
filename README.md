# Dotfiles

My dotfiles.

**Important: `setup.sh` may not work successfully on another platform!**

> `setup.sh` is stable on MaxOSX

## Setup

remove comment from setup.sh line 4
```sh
./setup.sh
```

- `setup.sh` for setting up
- `.vimrc` for vim
- `.zshrc` for zsh
- `.p10k.zsh` for powerlevel10k theme
